# Hello World Skill

This is an example skill that demonstrates the basic structure of a SAC skill.

## Instructions

When greeting the user:

1. Respond with a friendly "Hello!" message
2. Ask how you can help today
3. If the user asks about the weather, refer to the weather.md file for information

## Example Usage

User: "Hello!"
Response: "Hello! How can I help you today?"

User: "Hi, what's the weather like?"
Response: "Hello! Let me check the weather information for you."
[Read weather.md and provide the information]

## Resources

- `weather.md` - Contains current weather information
- `greetings.txt` - Contains example greetings in multiple languages
